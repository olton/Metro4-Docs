Initialisation components

About

Metro 4 support two ways to initialisation components: with data-role attribute, and with calling component initialisation function.

Using component role

The first way to initialisation components - using component role. To init component with role, you must add attribute data-role to element, and set component rolename as value. Options will be added with special data-* attributes. 


                    <div data-role="accordion">
                        ...
                    </div>

Using calling function

The second way to initialisation components - using calling special function. To init component with function, you must call Metro.makePlugin(...) method. This method receive three arguments: selector, component name, and component options. 


                    -- HTML
                    <div id="accordion">...</div>

                    -- JavaScript
                    var accordion = Metro.makePlugin("#accordion", "accordion", {...});
