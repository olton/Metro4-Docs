M4Q Visibility

You can control element visibility with special visibility functions.

About

You can control element visibility with special visibility functions: toggle(), hide(), show(), visible().

toggle()

To toggle element display property, you can use function toggle(). 


                    <button class="button" onclick="$('#div-toggle').toggle()">Toggle</button>
                    <div id="div-toggle"></div>
hide(), show()

To hide element (set display to none), you can use function hide(). To show element (set display to your initial), you can use function show(). 


                    <button class="button" onclick="$('#div-hide').hide()">Hide</button>
                    <button class="button" onclick="$('#div-hide').show()">Show</button>
                    <div id="div-hide"></div>
visible()

To toggle element visibility property, you can use function visible(). 


                    <button class="button" onclick="$('#div-visible').visible(false)">Hidden</button>
                    <button class="button" onclick="$('#div-visible').visible(true)">Visible</button>
                    <div id="div-toggle"></div>
Callback

With each visibility functions, you can use your own callback function. This function will be executed when the visibility function works done. 


                    <button class="button" onclick="$('#div-toggle').toggle()">Toggle</button>
                    <div id="div-toggle"></div>
